# CSL Styles Collection

A collection of Citation Style Language (CSL) styles for academic and professional citations.

## Styles Included

### Editora UFTM - Nota
A note-based citation style designed for publications from UFTM Editora (Federal University of Triângulo Mineiro). This style uses Portuguese (Brazil) localization and is suitable for books, book chapters, journal articles, conference papers, legal cases, and theses.

- **Format:** Note-based citations with bibliography
- **Language:** Portuguese (Brazil)
- **Author:** Rafael Dias Campos
- **License:** CC BY-NC-SA 4.0

### Editora UFTM (In-text)
Author-date citation style for UFTM Editora publications.

- **Format:** In-text author-date citations
- **Language:** Portuguese (Brazil)
- **License:** CC BY-NC-SA 4.0

### FCSH-UNL History Department
Historical reference style for FCSH-UNL (School of Social Sciences, New University of Lisbon) history department publications was the base of this CSL style.

- **Format:** Note-based citations
- **Language:** Portuguese
- **License:** CC BY-NC-SA 4.0


## Installation

### Zotero
1. Download the `.csl` file
2. In Zotero, go to **Edit → Preferences → Cite → Styles**
3. Click the "+" button and select the downloaded file

## Documentation
